# Life Expectancy Regression Analysis 🌍📈

## Project Overview
This project focuses on analyzing and predicting life expectancy using machine learning regression techniques and exploratory data analysis (EDA).

The dataset contains various health, economic, and social factors that influence life expectancy across different countries.

The project aims to discover important relationships between features and build predictive regression models to estimate life expectancy.

---

## Objectives
- Perform data cleaning and preprocessing
- Handle missing values and feature selection
- Explore correlations between health and economic indicators
- Apply regression techniques for prediction
- Evaluate model performance using regression metrics

---

## Dataset
Dataset: Life Expectancy Data (Kaggle)

The dataset includes:
- Health indicators
- Economic factors
- Mortality rates
- Vaccination statistics
- BMI and nutrition factors
- GDP and schooling information
